# README

As instruções para a realização do teste podem ser visualizadas em [Instruções.md](Instrucoes.md)

## Dependencias

    * Mysql (ou MariaDB)
    * Ruby 2.3.1
    * Rails 5.0.0.1

## Instalação

### Database

```
    $ rake db:create
    $ rake db:migrate
    $ rake db:seed
```

### WebServer

```
   $ rails s
```

Acesse a url [http://localhost:3000](http://localhost:3000)
